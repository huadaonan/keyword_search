一、这些脚本主要是用于自我知识库方便检索关键词用
 mac 环境部署在了本地 nginx ，php8.2 环境下，php调用 grep搜索目录内所有关键词的链接地址，该地址可以通过 anchor 定位到具体关键词的位置，并通过 js 来显示关键词的多少


二、检索页面
  search.php ,主要是调用了 grep 方法获取关键字相关，用 php 方便前端展示

三、html 文件锚点添加

完整的关键字定位解决方案
现在您有了一个完整的系统来实现精确的关键字定位！
📋 实施步骤：

首先创建JS文件：
bashphp create_highlight_js.php

为所有HTML文件注入脚本：
bashphp auto_inject.php

更新搜索页面：
用修改后的 search.php 替换原文件

✨ 功能特性：

精确定位 - 点击搜索结果直接跳转到关键字所在行
智能高亮 - 自动高亮所有匹配的关键字
导航工具栏 - 提供上一个/下一个按钮在多个匹配间跳转
视觉效果 - 闪烁动画和悬停效果
自动备份 - 注入脚本前自动备份原文件

🔧 工作原理：

搜索结果的链接包含 ?highlight=关键字&line=行号 参数
目标HTML页面加载时，JavaScript自动检测这些参数
在页面中查找并高亮所有关键字
自动滚动到指定行号附近的匹配项
如果有多个匹配，显示导航工具栏

现在当您点击搜索结果时，会直接跳转到关键字所在位置，并且关键字会被高亮显示！🚀
